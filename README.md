Node Express Mongoose Shop
-------------------------

MongoDB needed

1. npm i bower -g
2. npm i
3. bower i
4. set process.env params (phone, SID, AUTH_KEY, email, smtp) for login
5. run mongod from console
6. run Linqubase.js for db filling
7. run server.js
8. Open http://localhost:8080/
9. Sign is as Admin
